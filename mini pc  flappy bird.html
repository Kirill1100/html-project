# By Kyrylo Flappy Bird and Mini pc

Flappy Bird Arduino

Welcome! In this project, we learned to create a simple but interesting game "Flappy Bird" on Arduino.

This game is suitable for both beginners and more experienced electronics and programming enthusiasts.

What We Need

1. Arduino UNO or another sum board

2. Display (OLED or LCD) in this example we will use (LCD - 2004A - V1.3)

3. Button

4. Connecting wires

5. Breadboard
